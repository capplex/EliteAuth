Cookie Notice

EliteAuth uses essential storage for authentication and service operation, not behavioral advertising.

Last updatedJuly 21, 2026
OverviewEssential storageThird partiesYour choicesChangesContact

This notice explains the limited browser storage and external resources used by EliteAuth.

1. Overview

EliteAuth uses limited browser storage to provide account sessions, remember necessary application state, and support core functionality. This page uses “cookies” as a general term that can include cookies, local storage, and similar browser technologies.

2. Essential cookies and storage

Authentication providers may store session information needed to keep you signed in and protect account access. The website may also store temporary interface or workflow state required for requested features. These technologies are necessary for the service to function and are not used to sell advertising profiles.

3. External resources

The website loads fonts from Google Fonts and, on authentication pages, the Supabase JavaScript client from jsDelivr. Those providers may receive standard network information such as IP address, request time, referrer, and user agent. The deployed authentication and database provider may also set essential session storage.

4. Your choices

You can block or delete cookies using your browser settings. Blocking essential storage may prevent registration, sign-in, dashboard access, or other requested functions. EliteAuth does not currently use optional advertising cookies.

5. Changes

This notice may be updated if the technologies used by EliteAuth change. The date at the top identifies the latest revision.

6. Contact

Questions about browser storage or account sessions can be sent to support@eliteauth.lol.

Questions about this policy?

Contact support@eliteauth.lol with cookie or session questions.
