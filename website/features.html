Features

Every tool you need to control access.

A complete view of the capabilities available to every EliteAuth account, with no premium tier required.

Application and license management

Create product workspaces, define required versions, customize key formats, bulk-generate licenses, and enable or disable access instantly.

  • Timed and lifetime durations
  • Start expiry on first activation
  • Bulk export to TXT or CSV
  • Search and pagination

Device binding and reset workflow

Bind a license on first successful activation and require a controlled reset before it can move to another device.

  • First-activation HWID binding
  • One-time reset links
  • Manual approve or reject flow
  • Reset request reasons and status

Signed, replay-resistant sessions

Receive a short-lived token and one-time challenge only after the SDK verifies the Worker’s Ed25519 signature.

  • Pinned response verification key
  • Fresh timestamp and nonce checks
  • One-time challenge rotation
  • Hashed session-token storage

Release integrity and integrations

Approve the exact SHA-256 of a release build and use the included signed SDKs across your software stack.

  • Per-application build-hash enforcement
  • Integrity and replay security events
  • Eight included SDK languages
  • Webhook configuration controls
