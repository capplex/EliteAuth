# SDKs

Official source clients are available under `sdk/` for:

- C#
- C++
- JavaScript
- TypeScript
- Python
- Java
- Go
- Rust

Each client uses the same Worker API and verifies the signed response format. Configure the API URL, App ID, application version, and pinned public verification key. Never embed a service-role key, private signing key, or application server secret in distributed software.
